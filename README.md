# Student Grade Tracker (Java)

This is a simple **console-based Java application** to manage student marks and calculate their results.  
I built this project to practice **Java, OOP concepts, arrays, and ArrayLists**.

---

## What this project does
- Stores student details like **ID and Name**
- Accepts marks for **5 subjects**
- Calculates:
  - Highest mark
  - Lowest mark
  - Average mark
- Allows:
  - Adding a student
  - Removing a student
  - Viewing a single studentâ€™s details
  - Viewing all students at once
